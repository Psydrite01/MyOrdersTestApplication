# MyOrdersTestApplication

A simple Android application for viewing and managing orders, developed as an internship assessment.

Demo video: https://drive.google.com/file/d/1HgT3kmu4krixXjmS1-yHinuSE3ptK81x/view?usp=sharing

## Features

- **Orders List**: View all orders with filtering options (All, Completed, Cancelled, Booked Again)
- **Search**: Search orders by Order ID or location
- **Order Details**: View vehicle type, date/time, price, pickup/drop addresses, and status
- **Actions**: Download invoices and rebook orders
- **Bottom Navigation**: Home, Orders, Payments, Account tabs (Orders tab is fully implemented)
- **Help Button**: Floating action button for assistance

## Technical Details

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Minimum SDK**: 21
- **Architecture**: Single-activity with Navigation Component
- **State Management**: Rememberable state primitives
- **Animations**: Animated navigation between tabs
